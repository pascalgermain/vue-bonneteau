# Vue Bonneteau

> A Vue.js Bonneteau game.

> http://pascalgermain.github.io/vue-bonneteau/

## Project setup
```bash
pnpm install
```

### Compiles and hot-reloads for development
```bash
pnpm dev
```

### Compiles and minifies for production
```bash
pnpm build
```

### Lints and fixes files
```bash
pnpm lint
```
